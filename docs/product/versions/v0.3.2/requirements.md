## 摘要

- **版本目标**: 建立全公司共享的碎片观点库，员工可录入观点并在 AI 仿写时选择引用，为后续 AI 生成文案提供"血肉素材"。
- **功能数量**: 2 个（F-004-1 + F-004-2）
- **前置依赖**: v0.3.1 已完成的 AI 工作台仿写占位阶段，v0.3.3 中将利用本版观点数据接入 AI 生成
- **本版交付**:
  - 观点库独立页面：全公司共享，支持录入（单条+批量）、搜索、分页浏览、删除
  - 仿写阶段工作台内嵌观点选择区：搜索 + 多选，选中状态保存到本地 state
- **后续迭代**: v0.3.3 承接 AI 仿写生成（F-005-1），届时将仿写窗口的观点选择结果传入 AI prompt

## 边界结论

### 范围内

- 独立的「碎片观点库」管理页（`/viewpoints`）
- 员工可添加观点（单条 + 批量换行分割），纯文本，限 500 字/条
- 观点列表全文搜索（防抖）、分页浏览（Cursor 分页，20条/页）
- 全公司同一组织内数据共享，全量可见
- 展示创建者姓名 + 创建时间
- 删除权限：自己创建的观点可自删；BRANCH_MANAGER 及以上可删除任意观点（软删除）
- AI 工作台仿写阶段（`AiWorkspaceRewriteStage`）内嵌观点选择区：inline 搜索 + 多选，已选结果存在前端 state

### 非目标

- 观点分类/标签管理
- 观点导出
- 本版不触发 AI 仿写生成（AI 接入留到 v0.3.3）
- 不跨组织共享

## 功能清单

### F-004-1: 碎片观点录入（P0）

**描述**: 员工录入碎片化观点到全公司共享的观点池。

#### 入口

- 顶部导航或侧边导航新增「观点库」入口，路由：`/viewpoints`
- 页面顶部有「添加观点」按钮，打开录入弹窗

#### 录入弹窗

- 标题：「添加观点」
- 多行文本框（`textarea`）：支持输入一条或多条观点，多条时每行一条
- 提交时按换行符分割，逐条入库（每条去首尾空白后校验）
- 字数限制：每条上限 500 字；超限时单条报错，其余正常入库
- placeholder：`输入观点内容，一行一条，支持批量录入`
- 提交成功后刷新列表，弹窗关闭

#### 约束

- 空行跳过，不入库
- 同一员工可录入重复内容（不做去重）
- 需要 EMPLOYEE 及以上权限

---

### F-004-2: 观点浏览、搜索与选择（P0）

**描述**: 全公司员工可在观点库页面浏览、搜索、删除观点；在仿写阶段可从观点库多选观点备用。

#### 观点库页面

- 展示全公司（同组织）所有观点，按创建时间倒序
- 每条展示：
  - 观点文本（超过 3 行省略折叠）
  - 创建者姓名 + 创建时间
  - 删除按钮（仅自己创建的；BRANCH_MANAGER 及以上可删任意）
- 支持全文搜索（输入关键词过滤，防抖 300ms）
- Cursor 分页，20条/页，滚动加载

#### 仿写阶段观点选择区

- 在 `AiWorkspaceRewriteStage` 内新增「选择观点」区域
- 显示 inline 搜索框 + 结果列表
- 支持多选勾选，顶部显示「已选 N 条」
- 选中状态保存在前端工作台 state（`selectedFragmentIds`），**本版不写入 DB**（v0.3.3 AI 生成时使用）
- 快捷操作：清空所选

## 数据模型（新增）

```
model Fragment {
  id              String       @id @default(cuid())
  content         String       @db.Text       // 观点文本，500 字业务层校验
  organizationId  String
  organization    Organization @relation(...)
  createdByUserId String
  createdByUser   User         @relation(...)
  createdAt       DateTime     @default(now())
  updatedAt       DateTime     @updatedAt
  deletedAt       DateTime?    // 软删除

  @@index([organizationId])
  @@index([createdByUserId])
  @@map("fragments")
}
```

## 验收标准

1. `/viewpoints` 页面存在，导航可进入，展示观点列表。
2. 点击「添加观点」打开弹窗，支持单条和多行批量录入，提交后列表刷新。
3. 每条观点展示文本、创建者、时间；超长文本折叠显示。
4. 搜索框可过滤观点列表，防抖 300ms 正常工作。
5. 创建者可删自己的观点；BRANCH_MANAGER 可删任意；删除后列表刷新。
6. 仿写阶段内嵌观点选择区可搜索并多选，显示「已选 N 条」。
7. 观点状态保存在工作台 state，切换视频后重置（本版不持久化选中状态）。
8. 视觉风格与现有页面一致（参考用户管理页面风格）。
