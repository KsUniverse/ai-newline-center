## 功能范围

### F-000-1: 用户登录 (P0)

> 摘自 PRD M-000

- **描述**: 用户通过账号密码登录平台
- **交互**: 登录页 → 输入账号密码 → 验证 → 进入主界面
- **数据**: User (账号、密码hash、角色、归属组织)
- **边界**:
  - 密码错误提示（不泄露账号是否存在）
  - 登录态有效期管理（JWT/Session）
- **验收标准**:
  1. 管理员创建的账号可正常登录
  2. 不同角色登录后看到对应权限的界面
  3. 密码错误时有明确提示

## 基础框架要求

本迭代需同时完成项目技术基础搭建：

- Next.js 15 项目初始化（App Router）
- PostgreSQL + Prisma ORM 配置
- 基础布局框架（侧边栏导航 + 主内容区）
- 认证中间件（JWT）
- Seed 脚本创建初始超级管理员账号
