# v0.2.1 Backend Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the v0.2.1 backend sync loop for Douyin accounts, including manual sync, scheduled sync, and persisted sync timestamps.

**Architecture:** Extend the current Route Handler → Service → Repository layering instead of introducing a parallel path. `SyncService` will orchestrate crawler calls and repository writes, while `scheduler.ts` and `instrumentation.ts` will bootstrap periodic jobs without leaking cron concerns into route handlers.

**Tech Stack:** Next.js 15 App Router, TypeScript strict mode, Prisma 6, Vitest 3, node-cron 3, Zod 3
