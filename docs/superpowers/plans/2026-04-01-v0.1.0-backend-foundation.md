# v0.1.0 Backend Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Initialize the Next.js 15 application in the current `feature/v0.1.0` branch and implement the v0.1.0 backend authentication foundation end to end.

**Architecture:** Keep the repository root as the application root, preserve the existing `docs/` tree, and implement backend concerns in the documented layers: `lib` for shared runtime utilities, `server` for repository/service logic, `app/api` for the NextAuth route, and `prisma` for persistence and seed data. Authentication uses Auth.js Credentials with JWT sessions and Prisma-backed user lookup.

**Tech Stack:** Next.js 15, TypeScript strict mode, Auth.js v5, Prisma 6, PostgreSQL, Zod, bcryptjs, pnpm
